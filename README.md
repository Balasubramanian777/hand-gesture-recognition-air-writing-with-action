Real-time hand gesture recognition for air-writing with automatic application and website launching using Python, OpenCV, MediaPipe, and Flask.
# Hand Gesture Recognition for Air-Writing with Action Integration

## 📌 Project Overview

This project allows users to write alphabets and digits in the air using their index finger. The system captures hand movements through a webcam, tracks the finger using MediaPipe, and recognizes the written character using a trained AI model. After recognition, it automatically performs predefined actions, such as opening applications or websites, enabling touchless interaction with the computer.

## 🛠️ Technologies Used
- Python
- OpenCV
- MediaPipe
- Flask
- TensorFlow / Keras
- HTML
- CSS
